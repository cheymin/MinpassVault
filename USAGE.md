## 项目介绍

MinpassVault 是一款现代化的密码管理器，具有以下特点：

- 🔒 **零知识加密**：所有数据在客户端加密，服务器无法查看您的密码
- 🎨 **美观界面**：现代化的 UI 设计，支持深色/浅色主题切换
- 🌐 **多语言支持**：支持中文和英文界面
- 🔐 **两步验证**：支持 TOTP 两步验证
- 📧 **邮箱验证**：登录时可选邮箱验证码
- ☁️ **WebDAV 备份**：自动备份密码到 WebDAV 服务器
- 🔍 **泄露检测**：基于 Have I Been Pwned API 检测密码是否泄露
- 🧩 **浏览器扩展**：支持自动填充和自动保存密码

## 快速开始

### 1. 环境要求

- Node.js 18+ 
- npm 或 yarn
- Supabase 账户（用于数据库和认证）

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

复制 `.env.example` 为 `.env.local` 并填写配置：

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

### 4. 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:3000` 开始使用

### 5. 构建生产版本

```bash
npm run build
npm start
```

## 功能特性

### 密码管理

- **添加密码**：点击"+"按钮添加新的密码条目
- **分类管理**：支持登录信息、安全笔记、卡片、身份四种类型
- **密码强度检测**：自动分析密码强度
- **一键复制**：快速复制用户名和密码
- **密码生成器**：生成高强度随机密码

### 安全功能

#### 两步验证 (2FA)

1. 进入设置页面
2. 点击"启用两步验证"
3. 使用 Authenticator 应用扫描二维码
4. 输入验证码完成设置

#### 邮箱验证

1. 配置 SMTP 或 Resend 邮件服务
2. 启用邮箱验证
3. 登录时需要输入邮箱验证码

#### 数据泄露检测

1. 进入"数据泄露检测"页面
2. 系统会自动检查所有密码是否在已知泄露事件中
3. 使用 k-anonymity 技术保护隐私

### WebDAV 自动备份

1. 进入设置页面
2. 配置 WebDAV 服务器地址、用户名和密码
3. 启用自动备份
4. 系统会定期备份密码到 WebDAV 服务器

### 主题和语言

- **主题切换**：支持深色/浅色主题，默认深色
- **语言切换**：支持中文/英文
- **自定义字体**：支持网络字体加载

## 浏览器扩展安装

### 1. 准备扩展文件

扩展位于 `extension/` 目录

### 2. 加载到浏览器

#### Chrome/Edge

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `extension` 文件夹

#### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `extension/manifest.json`

### 3. 配置扩展

1. 点击扩展图标打开弹出窗口
2. 输入服务器地址（如：`http://localhost:3000`）
3. 输入用户名和密码登录
4. 登录成功后会自动保存

### 4. 使用扩展

- **自动填充**：访问登录页面时，扩展会自动填充保存的凭据
- **保存密码**：登录新网站时，扩展会提示保存密码
- **手动填充**：点击扩展图标，选择要填充的账户

## 常见问题

### Q: 重置密码后数据会丢失吗？

A: 是的。由于使用零知识加密，主密码是解密的唯一密钥。重置密码后，之前的数据将无法解密。请务必记住主密码或定期备份。

### Q: 如何迁移数据？

A: 支持 CSV 导入/导出：
1. 导出：在设置中点击"导出为 CSV"
2. 导入：在设置中点击"导入 CSV"

### Q: WebDAV 备份失败怎么办？

A: 检查以下几点：
- WebDAV 服务器地址是否正确
- 用户名和密码是否正确
- 服务器是否可访问
- 查看浏览器控制台错误信息

### Q: 扩展无法自动填充？

A: 确保：
- 扩展已登录
- 当前网站与保存的网址匹配
- 页面已完全加载
- 尝试刷新页面

### Q: 如何更换字体？

A: 在设置页面：
1. 点击"外观"部分的"展开"
2. 选择"默认字体"使用内置字体
3. 或选择"自定义字体链接"输入网络字体 URL

### Q: 两步验证丢失了怎么办？

A: 如果无法访问两步验证设备：
1. 使用备用验证码（如果保存了）
2. 联系管理员重置
3. 或重置账户（会丢失数据）
