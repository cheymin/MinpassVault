## 版本 2.1

### 📝更新内容

#### 新增功能
- 📱 手机端响应式设计 - 完美适配移动设备，支持各种屏幕尺寸
- ⏰ 15分钟登录时效 - 登录会话15分钟后自动过期，增强安全性
- 🔔 提示弹窗系统 - 操作成功/失败时显示美观的通知提示
- 🎨 UI全面美化 - 渐变背景、发光效果、动画过渡、现代化设计
- 🎨 Font Awesome 图标 - 使用专业图标库替换 emoji，提升视觉一致性
- 📱 手机端汉堡菜单 - 移动端添加三横线菜单，访问所有功能
- 📄 CSV 导出导出 - 支持浏览器密码管理器兼容的 CSV 格式
- 🔌 RESTful API - 提供 API 接口，支持第三方应用集成
- 📧 SMTP 邮件功能 - 支持发送验证码和密码重置邮件
- 🔑 忘记密码功能 - 用户可以通过邮箱重置密码
- 📄 密码重置页面 - 美观的重置密码界面
- 📧 邮箱验证功能 - 支持邮箱验证码发送
- 🎨 邮件模板 - 专业的 HTML 邮件模板设计

#### 改进内容
- 所有页面适配移动端（登录、初始化、保险库、设置）
- 按钮添加渐变背景和悬停发光效果
- 输入框使用半透明背景和聚焦动画
- 卡片组件添加悬停动画和阴影效果
- 模态框使用毛玻璃效果
- 滚动条使用渐变色设计
- 为设置页面各区域添加图标
- 优化颜色方案和视觉层次
- 导入导出功能增强 - 支持 JSON 和 CSV 双格式
- 图标系统统一 - 全站使用 Font Awesome 图标
- 弹窗样式优化 - 渐变背景、毛玻璃效果、自定义滚动条
- 登录页面添加"忘记密码"链接
- 数据库添加邮件相关字段支持
- API 路由优化，延迟初始化 Supabase 客户端
- 新增页面使用 Suspense 避免构建错误

#### 配置
- 添加 SMTP 邮件配置选项（可在设置中配置）
- 创建数据库迁移文件支持邮件功能
- 在初始化页面添加数据库配置
- 在设置页面添加 SMTP 配置
